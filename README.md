Logging module.

## Usage

### Install

```console
npm install teeverlog

pnpm install teeverlog

bun install teeverlog

yarn add teeverlog
```

### Initializing

```js
import 'teeverlog'

console.log('Hello, World!')
```

```console
[May 7, 12:25:38.065 AM] [LOG]: Hello, World!
```
